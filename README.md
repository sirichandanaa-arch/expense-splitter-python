# Smart Expense Splitter

A Python CLI application that simplifies shared expense management for groups by calculating balances and minimizing settlement transactions using a greedy algorithm.

## Features
- Equal and weighted expense splitting
- Net balance calculation for each participant
- Greedy algorithm to minimize settlement transactions
- Command-line interface for easy interaction

## Tech Stack
Python

## How to Run

1. Clone the repository
git clone https://github.com/sirichandanaa-arch/expense-splitter-python.git

2. Navigate to the folder
cd expense-splitter-python

3. Run the program
python main.py
